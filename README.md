# Airbnb Clone

## Features
- User authentication
- Add/Edit/Delete listings
- Image upload (Cloudinary)
- Reviews system

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Cloudinary
- EJS 

## Live Demo
👉 https://airbnb-clone-gules-nine.vercel.app/

## Setup Instructions
npm install
node app.js
